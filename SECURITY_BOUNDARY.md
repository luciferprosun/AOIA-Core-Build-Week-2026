# Security boundary

> **IN PROGRESS — NOT FINAL SUBMISSION EVIDENCE**

- This repository is evidence/staging only and contains no AOIA execution authority.
- Provider or model output is untrusted and non-authoritative.
- Tests, hashes, previews, route metadata, registries, and successful validation do not grant permission.
- No Hat is activated by this repository.
- No raw legal or Web Engineering corpus is stored here.
- No shell, browser, provider, download, GitHub, or write action may be inferred from a demo response.
- Any later demo must stay lightweight, bounded, and controlled by the human owner.
- Source worktree changes outside committed Git history are excluded.
- Private keys, tokens, passwords, environment files, private emails, personal case data, archives, and raw recordings are prohibited.
- Secret scanning is heuristic and cannot prove absence of every secret; human review remains required.

The committed AOIA-Core write kill-switch hardening is evidence of a source change, not authority conferred on this repository.
