# Report import policy

> **IN PROGRESS — NOT FINAL SUBMISSION EVIDENCE**

`accepted/` contains only small, committed, text-based Build Week evidence selected for direct relevance and checked for obvious secrets and personal case data. Every copied report has adjacent provenance metadata with source commit/path and SHA-256 values.

`pending/` contains status markers instead of partial reports. Raw corpora, archives, transcripts, provider logs, environment files, and unrelated historical reports are prohibited.
